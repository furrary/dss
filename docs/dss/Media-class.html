Media class

A media rule builder.

Media notForIpad() => Media()
  ..not()
  ..screen()
  ..minWidth('768px')
  ..maxWidth('1024px');

final box = new Dss()
  ..add('color', 'red')
  ..media(notForIpad()
    ..add('color', 'blue'));

The box will be red on an iPad, and blue on other media.
