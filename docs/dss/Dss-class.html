Dss class

An abstraction over a CSS rule.

This utilizes the builder pattern, so when you finish building your rule, you need to call register to make this rule usable.

You can specify this rule selector by calling select. If you don't specify the selector, it will be auto-generated as a class selector. You may access the generated uniqe class name via the getter name. Additionally, name will conveniently call register for you.

Please note that register can be called only once. Call it only when you finish building your rule.

Here's how to apply the rule.

querySelector('someSelector').className = (new Dss()
      ..add('color', 'red')
      ..add('font-size', '12px'))
    .name;

For a global rule:

new Dss()
  ..select('html, body')
  ..add('padding', '0')
  ..register();
