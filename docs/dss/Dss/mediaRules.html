mediaRules property

Map<String, List<String>> mediaRules
read / write

An abstraction over CSS @media rules.

This is a low level API. You might not need to access this.

This field can be null.

new Dss()
  ..select('p')
  ..add('font-size', '20px')
  ..mediaQueries = {
    '(max-width: 512px)': [
      'font-size: 10px',
    ]
  }
  ..register();

Compiles to:

p {
  font-size: 20px;
}

@media (max-width: 512px) {
  p {
    font-size: 10px;
  }
}
